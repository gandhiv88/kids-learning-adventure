# Kids Learning Adventure

A local-first, touch-friendly eight-question math adventure built with Next.js, React, TypeScript, and Tailwind CSS.

## Milestone 1 flow

Choose an original unicorn, robot, or dragon companion, then complete three addition, two subtraction, one missing-addend, one skip-counting, and one number-bond question. The companion changes placement and pose across questions and reacts gently to correct and retry feedback.

Question content remains deterministic. Answer order is generated by a seeded shuffle: every question has four distinct choices, the numeric correct answer appears once, and an eight-question session assigns the correct answer to each button exactly twice. A new session uses a new controlled seed, while tests can provide a fixed seed.

Progress is stored only in `localStorage`. For `math-foundations-1`, the app saves the best score and adds to cumulative adventure stars only when that best improves. Replaying an equal or lower score adds no stars. Existing version 1 progress safely migrates by preserving its selected character and cumulative total while starting this lesson's independently tracked best score at zero.
